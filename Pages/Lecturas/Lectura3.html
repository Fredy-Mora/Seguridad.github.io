Analisis de Riesgos

Es un proceso donde se realiza la identificación de los activos informáticos, las vulnerabilidades y amenazas a los que se encuentran expuestos, así como si probabilidad de ocurrencia y el impacto de estas, a fin de determinar los controles adecuados para aceptar, disminuir, transferir o evitar la posible ocurrencia de incidentes de este tipo.

Elementos de un análisis de riesgo: Durante este proceso de análisis de riesgo se genera normalmente un documento al cual se le conoce como matriz de riesgos. En este documento todos y cada uno de los elementos encontrados, la manera en que se relacionan y los cálculos realizados. Este análisis es vital para lograr una correcta administración de la administración del riesgo.

Análisis de impacto del negocio: Este análisis se realiza para poder establecer las prioridades, el sistema de gestión necesita saber el valor de los sistemas de información que pueden llegar a ser potencialmente afectados por incidentes de seguridad. Dentro de los valores para el sistema se puede distinguir: confidencialidad de la información, la integridad y finalmente la disponibilidad del sistema.

Ingenieria Social

Se conocen distintos tipos de ataques en internet como virus, troyanos u otros; estos ataques pueden ser contrarrestados, pero existe un tipo de ataque que no afecta directamente a los ordenadores llamado el eslabón mas débil, este se encarga de atacar a los usuarios siendo capaz de conseguir resultados similares a un ataque a través de una red, soltándose toda la infraestructura creada para combatir programas maliciosos. Este tipo de ataque es más eficiente debido a que es mas complejo de calcular y prever. Se pueden utilizar muchas influencias psicológicas para lograr que los ataques a un servidor sean lo mas sencillo posible, ya que el usuario inocentemente estaría dándole autorización para que dicha inducción se lleve a cabo.
